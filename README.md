# Predictia castigatorului unui meci de tenis folosind Machine Learning

## Descriere
Acest proiect isi propune sa prezica castigatorul unui meci de tenis folosind date statistice din ATP. 
Ideea este de a analiza informatii despre jucatori si meciuri pentru a observa daca se poate determina rezultatul unui meci folosind algoritmi de Machine Learning.
Proiectul va avea la baza analiza unor factori precum diferenta de ranking intre jucatori, forma acestora si rezultatele anterioare. 
  Pe baza acestor informatii se va incerca construirea unui model care sa faca predictii cat mai corecte.

## Sursa datelor
Datele vor fi preluate de pe site-uri publice care ofera statistici despre meciuri ATP, precum:
- https://www.kaggle.com/datasets/sijovm/atpdata
- https://www.atptour.com/en/stats/stats-home  
- https://www.tennisexplorer.com  

## Datele folosite
- statistici despre jucatori (ranking, rezultate, performanta)
- head-to-head intre jucatori
- rezultate ale meciurilor (castigator, scor, turneu, suprafata)
- analiza performantei jucatorilor

## Obiectiv
Scopul proiectului este de a construi un model de Machine Learning care sa poata prezice castigatorul unui meci de tenis. 
Pentru acest lucru vor fi analizate date precum rankingul jucatorilor, rezultatele meciurilor si alte statistici relevante, pentru a observa ce factori influenteaza cel mai mult rezultatul unui meci.
